# Cykl życia oprogramowania – SDLC

Projekt edukacyjny przedstawiający podstawowe zagadnienia związane z cyklem życia oprogramowania (Software Development Life Cycle – SDLC). Strona została wykonana w technologii HTML5 oraz Bootstrap 5 i ma charakter dydaktyczny.

---

## Opis projektu

Aplikacja prezentuje najważniejsze informacje dotyczące procesu tworzenia oprogramowania, w tym:

- definicję SDLC,
- etapy tworzenia systemu informatycznego,
- modele cyklu życia oprogramowania,
- podstawy podejścia DevOps,
- znaczenie testowania i wdrażania aplikacji.

Projekt został przygotowany jako materiał edukacyjny wspomagający naukę projektowania i wytwarzania oprogramowania.

---

## Technologie wykorzystane w projekcie

Projekt wykorzystuje następujące technologie:

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript (Bootstrap Bundle)
